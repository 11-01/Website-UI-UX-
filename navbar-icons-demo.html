🏷️ alt提升无障碍 | 📏 width/height防CLS | ⚠️ 非矢量缩放受限

🎨 方式三：CSS ::before 背景图 HTML极简

.nav-icon::before { background-image: url(...) } → HTML无图标标签，hover可切换

📊 三方案多维对比
维度字体图标<img>CSS ::before
可缩放性 ✅ 矢量，无损缩放 ⚠️ 受原图分辨率限制 ✅ 取决于素材格式
颜色修改 ✅ color/CSS一键 ❌ 需替换图片文件 ✅ 换SVG或CSS filter
Hover切换 ✅ color+transform ⚠️ 需两张图片 ✅ 换background-image
SEO/无障碍 ⚠️ 需加aria-label ✅ alt属性天然支持 ❌ 无文本替代
加载性能 ✅ CDN缓存，单文件 ⚠️ N个HTTP请求 ✅ 无额外请求
个性化程度 ⚠️ 受限于图标库 ✅ 完全自定义 ✅ 完全自定义
学习成本 ✅ 极低，查文档即可 ✅ 极低，标准HTML ⚠️ 中等，需CSS经验

📝 三种写法代码一览

<!-- 1. 在 <head> 中引入 Font Awesome CDN -->
<link rel="stylesheet"
      href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.1/css/all.min.css">

<!-- 2. 在导航链接中使用 <i> 标签 -->
<nav>
  <ul>
    <li><a href="#">
      <i class="fas fa-home"></i> 首页
    </a></li>
    <li><a href="#">
      <i class="fas fa-user"></i> 关于我们
    </a></li>
    <li><a href="#">
      <i class="fas fa-envelope"></i> 联系我们
    </a></li>
  </ul>
</nav>

/* 3. 悬停变色 */
nav a:hover i { color: #3b82f6; }
<!-- 使用 <img> 标签，务必加 alt/width/height -->
<nav>
  <ul>
    <li><a href="/home">
      <img src="icon-home.png"
           alt="首页"
           width="20"
           height="20">
      首页
    </a></li>
    <li><a href="/about">
      <img src="icon-about.png"
           alt="关于"
           width="20"
           height="20">
      关于我们
    </a></li>
  </ul>
</nav>

/* 悬停切换图片 */
nav a:hover img {
  content: url("icon-home-hover.png");
/* HTML — 极简，只有类名 */
<a href="/home" class="nav-icon home">首页</a>

/* CSS — 伪元素插入图标 + hover切换 */
.nav-icon {
  display: inline-flex;
  align-items: center;
  gap: 8px;
}
.nav-icon::before {
  content: "";
  display: inline-block;
  width: 20px; height: 20px;
  background-size: contain;
  background-repeat: no-repeat;
  background-position: center;
}
.home::before {
  background-image: url('icon-home.svg');
}
.home:hover::before {
  background-image: url('icon-home-hover.svg');
}
/* 图标弹跳放大 */
nav a .fa-icon {
  transition: transform .3s cubic-bezier(.34, 1.56, .64, 1);
}
nav a:hover .fa-icon {
  transform: scale(1.25) translateY(-2px);
  color: #f59e0b;
}

/* 底部下划线滑入 */
nav a::after {
  content: ''; position: absolute;
  bottom: 6px; left: 50%;
  width: 0; height: 2px;
  background: #f59e0b;
  transition: all .3s ease;
  transform: translateX(-50%);
}
nav a:hover::after { width: 24px; }

/* 消息红点 */
.badge-dot {
  position: absolute; top: 6px; right: 4px;
  width: 7px; height: 7px;
  background: #ef4444;
  border-radius: 50%;
}
