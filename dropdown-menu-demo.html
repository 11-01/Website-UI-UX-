网站UI下拉菜单完全指南

下拉菜单是网站导航的核心组件。两种主流实现方式：纯 CSS 悬停展开（桌面端）与 JavaScript 点击展开（移动端友好）。

纯 CSS 实现（悬停展开）

CSS Hover Dropdown

无需 JavaScript，代码简洁，适合桌面端导航。核心原理：position: relative + position: absolute 定位，:hover 伪类控制显示。

HTML 结构

<div class="dropdown">
  <button class="dropbtn">下拉菜单</button>
  <div class="dropdown-content">
    <a href="#">链接 1</a>
  </div>
</div>

CSS 关键样式

.dropdown { position: relative; }
.dropdown-content {
  position: absolute;
  opacity: 0;
  visibility: hidden;
  transform: translateY(-8px);
  transition: all .25s;
}
.dropdown:hover .dropdown-content {
  opacity: 1;
  visibility: visible;
  transform: translateY(0);
}
💡 使用 opacity + visibility + transform 而非 display: none/block，因为 display 无法添加过渡动画。

JavaScript 逻辑
function toggleDropdown() {
  document.getElementById("myDropdown").classList.toggle("show");
}

// 点击菜单外部区域时关闭
window.onclick = function(event) {
  if (!event.target.matches('.dropbtn')) {
    // 关闭所有打开的菜单
    document.querySelectorAll('.open').forEach(d => d.classList.remove('open'));
  }
};
⚠️ 别忘了处理"点击外部关闭"，否则菜单一旦打开就无法关闭，体验极差。

:focus-within — 键盘与触屏兼容方案

Focus-Within

:focus-within 伪类在容器内任意元素获得焦点时触发，天然支持 Tab 键导航和屏幕阅读器，解决了纯 :hover 在移动端的痛点。

实时演示 — 试试用 Tab 键导航到此按钮并按 Enter
🏠 首页 📦 产品 💰 定价 📞 联系我们
← Tab 键聚焦按钮即可展开，
继续 Tab 可在菜单项间导航
核心 CSS
/* 容器内任意元素获得焦点时展开菜单 */
.dropdown:focus-within .drop-menu {
  opacity: 1;
  visibility: visible;
  transform: translateY(0);
}
✅ 最佳实践：同时使用 :hover 和 :focus-within，覆盖鼠标和键盘两种交互方式。

进阶 UI 设计建议

🎬 平滑过渡动画

使用 opacity + transform: scaleY() + transition，避免 display: none/block。

🏷️ 语义化标签

推荐使用 <nav>、<ul> 和 <li> 嵌套构建菜单，提升可访问性。

📱 移动端适配

响应式设计中为移动端采用 JS 点击展开，或使用 :focus-within 增强兼容性。
