Bootstrap 5<select class="form-select">轻量 · 仅样式 · 无JS交互

🐛 避坑清单

① ID唯一性：页面多个下拉框，必须保证每个 id 唯一，或用 class + querySelectorAll。

② 悬停断连：CSS下拉菜单中，父级和子级之间不能有像素级空隙（margin-top），否则鼠标移动时菜单会意外收起。

③ A11y：自定义下拉菜单必须补全 ARIA：aria-expanded="true/false" + aria-haspopup="listbox" + role="option"。

📡 JS 动态生成选项

const data = ['北京','上海','广州'];
const sel = document.createElement('select');
data.forEach(city => {
  const opt = new Option(city, city);
  sel.appendChild(opt);
});
document.getElementById('container').appendChild(sel);

适合选项需从接口动态获取的场景。
