性能优化与无障碍提示

Performance & A11y

🚀 硬件加速

对频繁触发的动画添加 will-change: transform, opacity，让浏览器提前将元素提升到合成层。动画结束后务必移除。

.animating { will-change: transform, opacity; }
.done { will-change: auto; }

♿ 尊重用户偏好

务必使用 @media (prefers-reduced-motion: reduce) 媒体查询。对偏好减少动画的用户，关闭或大幅简化动效。

@media (prefers-reduced-motion: reduce) {
  *,*::before,*::after {
    animation-duration: .01ms !important;
    transition-duration: .01ms !important;
  }
}

📊 高性能属性对照表

✅ 仅触发 Compositetransform · opacity
⚠️ 触发 Paintcolor · background · box-shadow · border-color
❌ 触发 Layout（重排）width · height · top · left · margin · padding

原则：动画优先使用 transform + opacity，避免修改触发重排的属性。
