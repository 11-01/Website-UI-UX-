✓ 初始 opacity:0; translateX(-10px)
✓ 展开后 opacity:1; translateX(0)

关键：.sidebar:not(.collapsed) .menu-text { opacity:1; }
📊 方案对比矩阵
方案视觉性能触发的渲染阶段代码量
A. 无处理 文字截断消失差 Layout (width变化) 0
B. 仅opacity 淡出，较生硬一般 Layout + Paint 3行
C. 完整方案✅ 淡出+滑动，丝滑最优 Composite only (GPU) 8行
D. 展开淡入 对称动画，完整最优 Composite only (GPU) 5行
🚫
绝对禁止：用 width / left / margin 来做滑动动画 —— 这些属性触发 Layout 重排，导致卡顿。
唯一正确：用 transform: translateX() 实现位移，用 opacity 实现淡入淡出。
📝 HTML 结构 & 核心 CSS
<!-- HTML：文字用 <span class="menu-text"> 包裹 -->
<a href="#"><span class="icon">🏠</span> <span class="menu-text">首页</span></a>

/* ===== 推荐方案 ===== */
.menu-text {
  opacity: 1;
  transform: translateX(0) translateZ(0);   /* translateZ(0) → GPU合成层 */
  transition: opacity .35s cubic-bezier(.25,.46,.45,.94),
              transform .35s cubic-bezier(.25,.46,.45,.94);
  will-change: opacity, transform;                /* 提前通知浏览器 */
  white-space: nowrap;
}
.sidebar.collapsed .menu-text {
  opacity: 0;
  transform: translateX(-14px) translateZ(0);
}

/* ===== 展开淡入：文字初始透明 ===== */
.menu-text {
  opacity: 0;
  transition: opacity .4s cubic-bezier(.25,.46,.45,.94) .08s; /* 延迟.08s */
}
.sidebar:not(.collapsed) .menu-text { opacity: 1; }
