Grid vs Flexbox 选型指南

Decision Guide

🟢 首选 CSS Grid

二维布局（行列同时控制）
→ 圣杯 · 仪表盘 · 卡片网格 · 瀑布流

核心：1fr · minmax() · auto-fit · span

🔵 首选 Flexbox

一维布局（单行/单列对齐）
→ 导航栏 · 侧边栏 · 分栏 · 图文左右 · 居中

核心：flex: 1 · space-between · align-items

📐 拥抱相对单位
多用 fr、%、vw/vh、rem，少用死像素 px
🪄 响应式神器
minmax() + auto-fit + auto-fill 让你少写 @media
⚠️ Firefox 独占
CSS 原生 grid-template-rows: masonry 仅 Firefox 实验性支持
