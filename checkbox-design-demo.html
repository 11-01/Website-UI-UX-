基础原生实现（最推荐，兼顾可访问性）

Native Checkbox

使用 <input type="checkbox"> 配合 <label>，确保点击文字也能触发选中，对屏幕阅读器友好。

标准原生复选框

正确 HTML 结构

<!-- 方式1: for + id 关联 -->
<input type="checkbox" id="agree">
<label for="agree">同意条款</label>

<!-- 方式2: label包裹input -->
<label>
  <input type="checkbox"> 同意条款
</label>
2

快速修改颜色 — CSS accent-color

Accent Color

如果不需要改变复选框形状，只想让主色调与品牌色一致，一行 CSS 即可搞定。

accent-color 演示

一行 CSS

input[type="checkbox"] {
  accent-color: #22c55e;
}

✅ 简洁，不改变原生形状。
⚠️ 部分旧浏览器不支持（IE全系不支持）。

完全自定义 UI 样式（纯 CSS 方案）

Custom CSS Checkbox

核心步骤：隐藏原生 <input>（用 opacity: 0 保留焦点能力）→ 用 CSS :checked 伪类 + 相邻兄弟选择器 + 控制自定义图标的显示。

样式变体演示

标准圆角（默认）

圆形（Radio风格）

大号复选框

卡片式选择

核心 CSS 代码
/* 1. 隐藏原生input但保留焦点 */
.custom-cb input { position: absolute; opacity: 0; }

/* 2. 画自定义复选框 */
.checkmark { width: 22px; height: 22px; border: 2px solid #d1d5db; border-radius: 6px; }

/* 3. :checked 状态 */
input:checked + .checkmark { background: #3b82f6; border-color: #3b82f6; }

/* 4. ::after 画对勾 */
input:checked + .checkmark::after {
  content: ''; border: solid white;
  border-width: 0 2px 2px 0;
  transform: rotate(45deg);
}

/* 5. 焦点可见（无障碍关键！） */
input:focus-visible + .checkmark { outline: 3px solid rgba(59,130,246,.3); }
⚠️不要用 display: none 隐藏原生 input，这会使其失去键盘焦点能力！使用 opacity: 0 或绝对定位移出视口。

Toggle 开关（复选框变体）

Toggle Switch

Toggle 开关演示

Toggle 核心 CSS

.toggle-track {
  width: 44px; height: 24px;
  background: #d1d5db;
  border-radius: 12px;
  position: relative;
}
.toggle-track::after {
  content: ''; width: 20px; height: 20px;
  background: white;
  border-radius: 50%;
  position: absolute; top: 2px;
  transition: all .3s;
}
input:checked + .toggle-track {
  background: #3b82f6;
}
input:checked + .toggle-track::after {
  left: 22px;
}
