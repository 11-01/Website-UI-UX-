📦 fieldset 语义化分组

🎯 性别   

用 <fieldset> + <legend> 包裹单选组，屏幕阅读器可理解"性别：男/女"的语义。

⚛️ 框架开发

<!-- Vue: v-model 双向绑定 -->
<input type="radio" v-model="picked" value="a">

// React: useState + onChange
const [picked, setPicked] = useState('a');

Vue: v-model · React: useState + onChange。数据驱动视图，避免直接操作DOM。

🛡️ 服务端校验

前端单选限制可被绕过（浏览器DevTools修改name属性即可多选）。服务端必须校验接收到的参数值是否合法且唯一。

🧩 UI 组件库

Element Plus: <el-radio-group v-model="val">
Ant Design: <Radio.Group onChange={...}>

企业项目推荐使用组件库，已内置 A11y、响应式、按钮组/卡片式等变体。
