无障碍设计 + 性能与一致性

A11y & Performance

♿ 语义化 HTML + ARIA

<nav aria-label="分页导航">
  <ul>
    <li><a href="?page=1" aria-current="page">1</a></li>
  </ul>
</nav>

使用 <nav> + aria-current="page" + aria-disabled，确保屏幕阅读器用户能理解分页状态。

⚡ 性能优化

分页能显著减少单次页面加载时间。后端配合：

SELECT * FROM items
ORDER BY created_at DESC
LIMIT 20 OFFSET 40; /* 第3页 */

使用 LIMIT/OFFSET + 合理索引，确保翻页响应迅速。

🔄 全局一致性

整个网站的分页样式、位置和行为必须保持统一。包括：页码尺寸、颜色主题、省略号位置、上一页/下一页文案。

📝 URL 与 SEO

分页应能通过 URL 参数定位：?page=3。SEO 方面使用 rel="prev" / rel="next" 链接标签告知搜索引擎页面关系。
