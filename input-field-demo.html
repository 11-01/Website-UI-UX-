❌ 输入后消失，用户容易忘记字段要求。

移动端适配与 HTML5 type 属性

Mobile & HTML5

📱 正确使用 type 调起对应键盘

🔒 HTML5 内置验证属性

<!-- 必填字段 -->
<input required>

<!-- 正则模式 -->
<input pattern="[0-9]{6}" title="6位数字">

<!-- 长度限制 -->
<input minlength="6" maxlength="20">

结合 JS 实时校验提供更好的用户体验。

⚠️小屏幕下输入框应设置 width: 100% 自适应。字号 ≥ 16px 可防止 iOS Safari 自动缩放页面。
5

安全防护与进阶功能

Security & Advanced

🛡️ XSS 防护 - 输入转义

function sanitize(input) {
  const div = document.createElement('div');
  div.textContent = input;
  return div.innerHTML;
}

使用 textContent 赋值而非 innerHTML，自动转义恶意脚本。

🧠 智能提示与格式化

输入邮箱"@"后自动提示常用域名（@qq.com、@gmail.com）；输入手机号/信用卡号时自动添加空格分隔。

♿ 无障碍 (A11y)

色彩对比度 ≥ 4.5:1（WCAG AA）。使用 <label for="id"> 关联输入框。错误信息使用 aria-describedby。支持纯键盘 Tab 导航。

🧩 企业级组件库

Ant Design · TDesign · Bootstrap 已内置响应式、无障碍和状态交互规范。推荐在企业项目中使用，保证一致性并大幅提升开发效率。
